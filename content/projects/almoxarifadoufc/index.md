**Warehouse Control System** developed for the Federal University of Cear치 - Quixad치 Campus to automate and streamline the equipment loan process.

## Problem

Manual equipment loan management at UFC Quixad치 campus was inefficient, lacked proper tracking, and required physical presence for requests.

## Solution

This system addresses these issues by providing:

- **Automated lending process** - Eliminates manual paperwork and streamlines operations
- **Remote accessibility** - Users can request equipment loans from anywhere with internet access
- **Inventory management** - Tracks equipment availability and usage in real-time
- **Administrative workflow** - Request approval system with automated notifications
- **Comprehensive reporting** - PDF generation for inventory, loans, and usage history
- **Document validation** - Hash-based authentication ensures document authenticity
- **Analytics dashboard** - Charts and visualizations for better decision-making

## Key Features

- Equipment loan request management
- Administrative approval workflow
- PDF report generation (inventory, loans, history)
- Email notification system (PHPMailer)
- "Nada Consta" (no pending items) certificate generation
- Dashboard with Chart.js analytics
- User blocking system for violations
- reCAPTCHA security integration

## Technologies

Built with **PHP**, **MySQL**, **HTML**, **CSS**, and **JavaScript**. Includes TCPDF for PDF generation, PHPMailer for notifications, and Chart.js for data visualization.

## Impact

Significantly improved operational efficiency for the campus warehouse, reduced processing time for equipment loans, and provided better visibility into inventory and usage patterns.
